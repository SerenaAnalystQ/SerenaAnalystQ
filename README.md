## [Plug Tech E-Commerce Performance Analysis](https://github.com/SerenaAnalystQ/Plug_Tech_E-commerce_Sales_Analysis)

This project analyzes Plug Tech’s e-commerce sales performance from 2019 to 2022 using SQL, Excel, and Tableau. The analysis focused on revenue trends, order volume, average order value, regional performance, product contribution, refund behavior, and loyalty program effectiveness. The dataset included over 108,000 transactions and nearly 88,000 customers across multiple global regions. Key findings showed that revenue peaked in 2020 due to strong demand for premium electronics, but declined sharply in 2021–2022 as order volume and AOV weakened. North America generated 52% of total revenue, creating geographic concentration risk, while Japan showed strong premium-market potential with the highest AOV. Final recommendations included optimizing seasonal campaigns, expanding underpenetrated markets, improving loyalty engagement, and managing high-refund products. View the project report [here](https://github.com/SerenaAnalystQ/Plug_Tech_E-commerce_Sales_Analysis).

## [Prime Care Marketing Analysis](https://github.com/SerenaAnalystQ/Prime_Care_Marketing_Analysis)


This project analyzes 50K+ healthcare marketing records to evaluate campaign performance, customer acquisition efficiency, and budget allocation across 12 campaigns. The analysis delivered a self-serve Tableau dashboard and slide deck to report key marketing and signup metrics to Prime Care’s Marketing Team. By identifying resource reallocation opportunities, the insights helped improve Click-Through Rate by 17% and Signup Rate by 13%.
